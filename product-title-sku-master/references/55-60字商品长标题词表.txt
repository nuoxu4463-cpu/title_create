标题目标：
商品 + 规格 + 核心功效 + 症状长尾词 + 人群/场景词

使用原则：
1. 前18-25字必须让系统识别“卖的是什么”，不要一上来全是症状词。
2. 中段放核心功效/适用方向，例如清洁护理、体重管理、营养补充、屏障修护。
3. 后段堆症状长尾词，尽量覆盖部位、人群、场景、痛点。
4. 你不能做OTC药品，所以标题里尽量用“护理、营养、管理、舒缓、清洁、修护、辅助”这类合规词。
5. 避免直接写“治疗、治愈、根治、药效、处方、消炎杀菌、快速见效”等高风险词。

标题组合公式：

公式1 标准扩流型
商品名 + 规格 + 核心功效 + 症状1 + 症状2 + 部位 + 人群 + 场景

例：
女性私密护理凝胶5支装 + 清洁抑菌 + 瘙痒异味白带异常 + 干涩敏感 + 经期日常护理

公式2 成分识别型
成分/材质 + 商品名 + 规格 + 功效方向 + 症状长尾 + 人群场景

例：
白芸豆压片糖果60片 + 体重管理餐前控热量 + 腰腹赘肉大肚腩小肚子 + 代谢慢

公式3 症状抢词型
商品名 + 规格 + 核心痛点 + 关联痛点 + 人群 + 使用场景 + 合规功效词

例：
生理盐水鼻喷100ml + 鼻塞鼻干鼻痒 + 过敏季换季 + 儿童成人 + 鼻腔日常护理

公式4 人群场景型
商品名 + 规格 + 人群 + 场景 + 核心功效 + 症状词组

商品名 + 规格 + 隐私/保密 + 核心功效 + 症状长尾 + 使用场景

例：
私密抑菌凝胶3支装 + 保密发货 + 女性护理 + 异味瘙痒干涩白带异常 + 敏感护理


高曝光标题的底层逻辑：
不是标题越短越好，而是系统能明确判断“这是什么商品、解决什么痛点、该推给谁”。

低曝光标题常见问题：
1. 只有品牌+规格，没有症状词。
2. 只有商品名，没有人群和场景。
3. 类目词太泛，例如“护理液、凝胶、喷雾、营养片”，系统不知道推给谁。
4. 症状词太硬，容易触碰药品/治疗表达，系统反而可能压流。
5. UPC、类目、标题、图片表达不一致，导致系统无法稳定归类。
